# Global Health Data Analysis

This project analyses global health indicators related to the United Nations Sustainable Development Goal 3 (SDG 3).

## Project Overview
The project investigates relationships between food insecurity, health expenditure, and health outcomes across multiple countries. It combines data preparation, exploratory data analysis, regression modelling, and clustering techniques.

## Objectives
- Analyse health-related patterns across 10 countries
- Explore relationships between food insecurity, health expenditure, and life expectancy
- Forecast selected health indicators using regression models
- Group countries with similar health patterns using clustering methods

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Methods Used

### Data Preparation
- Cleaned missing and inconsistent values
- Standardised country names and column formats
- Merged multiple datasets for analysis

### Exploratory Data Analysis
- Descriptive statistics
- Histograms
- Scatter plots
- Boxplots
- Cross-country comparison

### Machine Learning
- Linear Regression
- K-Means Clustering
- Hierarchical Clustering
- StandardScaler
- Elbow Method
- Silhouette Score

## Key Findings
- Higher government health expenditure was associated with better health outcomes
- Countries with higher food insecurity tended to have lower life expectancy
- Clustering helped identify countries with similar health indicator patterns
